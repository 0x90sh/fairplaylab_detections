# FairplayLab Detections

Runnable example code and demos for the FairplayLab Bible.

Read the full documentation at:

https://bible.fairplaylab.org/

## Repository Layout

* `docs/` - GitBook source files synced with the FairplayLab space.
* `usermode-process_handle/` - Process handle detection demo.
* `usermode-overlay/` - Overlay window detection demo.
* `usermode-ext_rw_process_mem/` - External process memory access detection demo.
